# Data Availability

This repository provides data-access information and supplementary materials for the manuscript.

Some raw datasets used in the study are subject to provider access restrictions, platform terms, or third-party licensing conditions. Therefore, users should obtain the original datasets from the official sources listed below and comply with the corresponding terms of use.

## Data Sources

| Data source | Access information |
| --- | --- |
| DiDi ride-hailing OD data | The core DiDi ride-hailing OD data should be requested through the DiDi GAIA research collaboration program. Access may be subject to application review and a non-disclosure agreement. |
| Amap POI data | Amap POI data should be obtained through the Amap Open Platform using the appropriate developer account, API key, and service interface. Users should follow Amap's developer terms and API usage rules. |
| GHS-POP population data | The GHS-POP 1-km gridded population surface is publicly available from the Global Human Settlement Layer (GHSL), European Commission / Joint Research Centre. |
| OpenStreetMap road network | OpenStreetMap road network data are publicly available from OpenStreetMap and regional extract providers such as Geofabrik. |

## Notes

- Restricted or third-party raw data are not redistributed in this repository unless redistribution is permitted by the original data provider.
- Users wishing to reproduce the analysis should obtain the source datasets from the official providers above and process them according to the methods described in the manuscript.
- Please cite the original data providers and comply with their licensing, access, and attribution requirements.
